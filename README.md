# Monochrome Motion Portfolio

This project was generated based on the `Portofolio.md` specification.

## Tech Stack
- ReactJS
- Vite
- Tailwind CSS
- Framer Motion

## Setup

1. Navigate to the project directory:
   ```bash
   cd portfolio
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run development server:
   ```bash
   npm run dev
   ```

4. Build for production:
   ```bash
   npm run build
   ```

## Structure
- `src/components`: UI Components (Stepper, LightRays, Sections)
- `src/pages`: Page layouts
- `src/data`: Data content
